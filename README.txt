== Description ==
**TSA Commerce** convierte tu WordPress en un completo "Turbo SEO Affiliate Commerce".
De forma fácil y sencilla. Sin necesidad de usar WooCommerce.

CARACTERÍSTICAS

* Fácil y sencillo de instalar.
* Shortcodes configurables.
* Gestión de Categorías propio.
* Gestión de Productos de Afiliación.
* Generador de enlaces internos fácil con shortcodes.
* Posibilidad de elegir el número de columnas para los productos.
* Sistema de grid a elegir: Propio, Bootstrap y Foundation.

== Installation ==
1. Subir la carpeta tsa-commerce al directorio /wp-content/plugins/ vía FTP.
1. Subir el archivo ZIP completo vía Plugins -> Añadir nuevo -> Subir en el Panel de Administración de tu instalación de WordPress.
1. Buscar TSA Commerce en el buscador disponible en Plugins -> Añadir nuevo y pulsar el botón Instalar ahora.

== Frequently Asked Questions ==
= ¿Cómo puedo mostrar productos en una página? =
Para mostrar los productos de una categoría en un página utiliza el shortcode [tsa_products category="slug-category"] 

= ¿Cómo puedo mostrar todo el listado de categorías? =
Para mostrar las categoría en un página utiliza el shortcode [tsa_categories]

= ¿Cómo puedo mostrar el listado de categorías relacionadas? =
Para mostrar las categoría relacionadas en un página utiliza el shortcode [tsa_related_categories categories="term1,term2,term3,..."]

= ¿Cómo se configura? =
En el menú lateral TSA Commerce -> Ajustes puedes añadir el ID de afiliado de Amazon (Amazon Associates ID) y si deseas que se muestre o no el precio.
